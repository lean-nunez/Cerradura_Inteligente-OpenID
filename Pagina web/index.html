🔐 Open ID - Cerradura Inteligente

Proyecto de Robótica y Programación - 5to Año

Introducción

Nuestro proyecto busca optimizar la seguridad en sistemas de control de acceso mediante el uso de tecnología RFID (tarjeta o tag) y autenticación por PIN. Sustituimos las llaves físicas por un sistema moderno y seguro que además puede integrarse con IoT y domótica.

Objetivo General

Diseñar y desarrollar una cerradura inteligente que incremente la seguridad y comodidad de acceso a espacios físicos, integrando RFID, PIN y pantalla LCD.

Objetivos Específicos

  • Implementar autenticación por tarjeta RFID
  • Agregar validación secundaria mediante código PIN
  • Mostrar información y estado del sistema en pantalla LCD
  • Optimizar seguridad y consumo energético

Materiales

  • Arduino UNO
  • Módulo RFID RC522
  • Pantalla LCD 16x2
  • Teclado matricial 4x4
  • Servomotor
  • Buzzer y LEDs de estado

Plan de Testing

#Tipo de prueba
1Lectura RFID

5Activación de cerradura

Probables Fallas

  • Fallo de lectura RFID por interferencias
  • Clonación de tarjetas RFID
  • Errores en la validación cruzada PIN + RFID
  • Consumo energético excesivo
  • Vulnerabilidades en almacenamiento de datos
